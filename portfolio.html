Quick Flips RS

Quick Flips RS is a hobby project for a game that I enjoy called Old School Runescape.

This site takes advantage of an API to access information from the in-game auction house called the "Grand Exchange". You can search for items and see their potential profit margins based on timing information. It also features an autofill search bar to make finding items easier.
